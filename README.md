FEATURES: 
Upload resumes in PDF format.
Extract technical skills using NLP (spaCy).
Compare resume skills with a job description.
Calculate a job match score using TF-IDF and Cosine Similarity.
Display matched and missing skills.
Recommend suitability for interview based on match score.

USAGE:
Run the Streamlit app:
CODE:
streamlit run app.py

IN THE WEB APP:
Upload a resume (PDF format).
Paste the job description.
Click Analyze Resume.
View the results:
Match Score (%)
Matched Skills

Missing Skills

Recommendation (Suitable / Needs Improvement)
